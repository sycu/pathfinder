# Pathfinder
Pathfinding algorithms with graph creator and visualizations

## Features

### Graph creators
- Matrix

### Algorithms
- A* (configurable heuristic)
- Best First Search
- Bidirectional A* (configurable heuristic)
- Breadth First Search
- Dijkstra

### Heuristics
- Euclidean
- Manhattan

## Usage
```python
python pathfinder.py [matrix_name] [cols] [rows]
```
